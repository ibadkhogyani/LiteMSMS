# LiteMSMS 
LiteMSMS is a Mobile Shop Management System ,written in C# with SQLite as the Database.

<h3>Features:</h3>
<ul>
<li>Sale Invoice</li>
<li>Sale Return</li>
<li>Purchase Invoice</li>
<li>Purchase Return</li>
<li>Sale Report</li>
<li>Purchase Report</li>
<li>Profit Report</li>
<li>Database Backup</li>
<li>Database Restore </li>
</ul>
